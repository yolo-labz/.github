02 / 06
supply chain

Every release, provable.

  • SLSA L2 build provenance — who built it, from which commit
  • Sigstore keyless signing — OIDC, no long-lived keys
  • Dual SBOM — CycloneDX 1.6 + SPDX 2.3
  • Reproducible builds — pinned SOURCE_DATE_EPOCH
  • SHA-pinned actions — every workflow step, by digest
03 / 06
verify it yourself

One command. No cosign install.

gh attestation verify <asset> --repo yolo-labz/<plugin>

✓ Confirms the artifact was built by our CI, from this source, and has not been tampered with since signing. The provenance travels with every published release asset.

04 / 06
how we ship

Hardened by default.

  • Workflow permissions deny-all — re-granted per job
  • harden-runner egress policy — monitored network
  • Tag-gated on green main — no release on red CI
  • Never re-tag — botched publish ships vX.Y.Z+1
  • Conventional commits — auto-generated changelog
